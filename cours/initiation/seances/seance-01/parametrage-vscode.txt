# Paramétrage de l'éditeur Visual Studio Code

1. [Télécharger Visual Studio Code](https://code.visualstudio.com/download)
2. Installer puis ouvrir l'éditeur
3. Dans le menu, aller à « View » puis sélectionner « Command Palette » (__CTRL/CMD+SHFT+P__)
4. Dans le panneau qui apparaît à l'écran, taper « Extensions » puis sélectionner « Extensions : Intall Extensions ».
5. À la gauche, dans le panneau des extensions, chercher « Live Server ».
6. Installer l'extension « Live Server »
7. Redémarrer l'éditeur.
8. Pour tester le fonctionnement l'extension, ouvrir _un dossier comprenant au moins un fichier_ de votre choix dans l'éditeur en allant dans le menu « File » et sélectionnant « Open ».
9. Quand votre dossier est ouvert dans l'éditeur, repérer la mention « Go Live », en bas à droite dans la barre d'état de l'éditeur.
10. Si le test est concluant, Google Chrome devrait s'ouvrir, présentant une vue « Index » du dossier choisi.
11. Bravo ! Visual Studio Code est maintenant prêt à être utilisé (avec une visualisation directe en prime).
